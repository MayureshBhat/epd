01 — Process

Five stages,
under 50 milliseconds.

Each frame from your camera travels through a pipeline that ends in a confirmed letter on screen.

01
📷
Webcam capture
720p frames grabbed live via the browser's getUserMedia API.
02
✋
Landmark detection
MediaPipe WASM extracts 21 3D points per hand, per frame.
03
📐
Feature engineering
Finger extension states and joint angles computed in JavaScript.
04
🧠
Classification
Pattern lookup maps the 5-finger binary string to an ASL or ISL character.
05
💬
Text output
A 7-frame stabiliser confirms gestures before adding them to the sentence.

03 — Capabilities

What the system can do.

F — 01
Indian Sign Language
Full Devanagari character support — vowels (स्वर), consonants (व्यंजन), and digits with context-aware disambiguation for ambiguous handshapes.
F — 02
American Sign Language
All 26 English letters. Pattern logic and angle checks separate look-alike handshapes such as C and S, or O and a closed fist.
F — 03
Runs entirely in the browser
MediaPipe compiles to WASM and runs client-side. No server, no Python, no installation. Open the URL and sign.
F — 04
Confidence scoring
Every prediction is scored across landmark spread, hand steadiness, and pattern exclusivity — low-confidence frames are filtered out.
F — 05
Word and sentence builder
A hold-to-confirm mechanic prevents accidental inputs. Letters accumulate into words; words accumulate into sentences.
F — 06
Finger state visualiser
Live indicator bars show which fingers the model detects as extended — so you can see exactly what the classifier is reading.
04 — Stack

Technology used.

MediaPipe Hands
Google's 21-point 3D hand landmark model, compiled to WASM. Same model used in the Python notebook — identical accuracy.
OpenCV + Python
Full-accuracy local pipeline with curvature-angle classification, session saving, and text-to-speech output.
NumPy
Vector arithmetic for joint angles, finger curvature scoring, and hand steadiness in the Python backend.
Vanilla JS
No framework. Direct Canvas 2D drawing, deque-style frame buffer, pattern lookup table, keyboard shortcuts.
